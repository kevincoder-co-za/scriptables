Logs

Whenever a build is triggered from any action such as setting up a server, deploying a site, adding cron jobs etc... - Scriptables will track every action being executed on your server.
You can then access, these logs by clicking on the event log icon as per below under actions.
For security reasons - logs are encrypted before storage, so you can cannot read them directly in the DB and will only be able to decrypt and view them from this log viewer in Scriptables.
To get more detailed logs in your terminal, you can also set this env: VERBOSE_LOGS=yes , which will appear in your docker logs.
